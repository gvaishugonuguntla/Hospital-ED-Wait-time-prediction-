# Hospital-ED-Wait-time-prediction-
Predicting emergency department waiting times using statistical modelling and machine learning.

## Overview
Emergency departments face increasing pressure due to rising patient demand and limited resources. Long waiting times negatively impact patient satisfaction and healthcare outcomes.

This project analyses emergency department operational data to identify factors affecting patient waiting times and develops predictive models to support hospital planning and resource allocation.

## Objectives
- Analyse patterns in emergency department patient arrivals
- Identify determinants of waiting time
- Build predictive models to forecast waiting times
- Support operational decision making

## Tools Used
- R
- RMarkdown
- Statistical modelling
- Data visualisation

## Dataset
The dataset used for this project contains confidential hospital patient records and cannot be shared publicly due to NHS data privacy regulations.

This repository therefore contains the full modelling workflow without the original dataset.
